## TL;DR

To connect to your crawling server, you need:

1. The server IP address (or hostname)
2. Your username and password
3. A remote desktop client installed on your machine

Most users on Windows can use **Microsoft Remote Desktop**.  
Mac users can use **Microsoft Remote Desktop** or **Jump Desktop**.  
Linux users can use **Remmina**.

## Remote Desktop Client Options

Here are common options by platform:

### Windows

- **Remote Desktop Connection** (built-in on most Pro/Enterprise installs)
- **Microsoft Remote Desktop** (Store app)
- **AnyDesk** (alternative workflow)

### macOS

- **Microsoft Remote Desktop**
- **Jump Desktop**
- **AnyDesk**

### Linux

- **Remmina**
- **KRDC**
- **AnyDesk**

## Basic Connection Steps

1. Open your chosen remote desktop app.
2. Enter the server IP or hostname.
3. Enter your username and password when prompted.
4. Accept any first-time certificate/security prompt.
5. Save the profile/connection for future sessions.

## Troubleshooting (Quick Checks)

- **Cannot reach server**: verify IP/hostname and that your network/firewall allows outbound RDP.
- **Credentials rejected**: confirm username format and reset password if needed.
- **Connection drops/lag**: reduce display resolution and disable heavy visual effects in the client.
- **Still blocked**: send us your error screenshot and the timestamp of your login attempt.
